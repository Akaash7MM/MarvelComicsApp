# Marvel Comics App

## General info
The Marvel Comics App allows the user to browse through various marvel comics
	

## Technologies
Project is created with:

* Kotlin
* Jetpack Compose
* Clean MVVM
* Ktor
* Dagger-HILT
* Coroutines
